# TaskForceAI Open

This repository contains the open client and developer platform surface for TaskForceAI.

## Included

- Web, mobile, desktop, TUI, console, status, docs, and marketing apps
- TypeScript, Python, Rust, and Go SDKs
- Client runtime, UI, persistence, sync, observability, contracts, and protocol packages

## Not Included

The backend control plane is intentionally private: auth, billing, core, developer, engine, sync, admin, model-server, evaluation, database, infrastructure, and service-internal packages.

See [OPEN_SOURCE.md](./OPEN_SOURCE.md) for the public boundary and contribution policy.
