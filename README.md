# TaskForceAI Open

This repository contains the active product implementation, clients, and developer platform for TaskForceAI.

## Included

- Web, mobile, desktop, TUI, console, status, docs, and marketing apps
- Auth, billing, core API, developer API, engine, and sync services
- Product rules and orchestration core in Go, Rust, Python, and TypeScript
- TypeScript, Python, Rust, and Go SDKs
- All reusable packages, including adapters, contracts, infrastructure, persistence, UI, and provider integrations

## Not Included

Credentials, production data, and live deployment state are not included.

See [OPEN_SOURCE.md](./OPEN_SOURCE.md) for the public boundary and contribution policy.
