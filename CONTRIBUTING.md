# Contributing

Thanks for contributing to TaskForceAI.

1. Install the Bun version pinned in package.json.
2. Run bun install.
3. Keep changes focused and include tests for behavior changes.
4. Run the relevant package checks before opening a pull request.
5. Run bun run security:check for security-sensitive changes.

Never commit credentials, .env files, production data, signing material, or generated build output. Security reports should follow SECURITY.md when present or be reported privately to the maintainers.
