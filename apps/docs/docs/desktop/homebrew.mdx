---
title: 'Homebrew Installation'
description: 'Install TaskForceAI Desktop via macOS Homebrew'
---

The recommended way to install TaskForceAI on macOS is using Homebrew Cask.

## Installation

Run the following command in your terminal:

```bash
brew tap ClayWarren/taskforceai && brew install --cask taskforceai
```

## Upgrading

Keep your app up to date with a single command:

```bash
brew upgrade --cask taskforceai
```

## Maintenance

Check your current installation status:

```bash
brew info --cask taskforceai
```
